# Electron Typescript Vue 项目模版

## 开始 
### 项目安装
```
yarn install
```

### 启动 Web 开发环境
```
yarn run serve
```
### 启动 Electron 开发环境
```
electron:serve
```

## 打包
### Web 端打包
```
yarn run build
```
### Electron 打包
```
yarn run electron:build
```

## 测试
### 运行测试
```
yarn run test
```

### 运行 Lints 和 修复文件
```
yarn run lint
```

### 运行 e2e 测试
```
yarn run test:e2e
```

### 运行单元测试
```
yarn run test:unit
```
